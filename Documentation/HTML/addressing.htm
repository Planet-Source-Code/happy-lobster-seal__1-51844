Addressing

This is a way of telling an instruction what data to use in its computation.

Indexed

This is where an array name is used as an address. Arrays are 1-dimensional and can be declared in any part of the program and must be declared.

Declaration is done as follows:

<ArrayName>: Blocki <SizeOfArray>

where <SizeOfArray> is an integer that represents the size of the array including element(0).

e.g. MyArray: Blocki 2 ;Declare an array called MyArray with 2 elements; MyArray(0) and MyArray(1).

Used:

<Instruction>  <Register>, MyArray(<Real>)
<Instruction>  <Register>, MyArray(<Register>)

Where <Real> is a positive integer and register <Register> = Acc | Indx

e.g. Load Acc, MyArray(2)

 

Direct Addressing

This is where a variable name is used as an address.  The variable can be declared in any part of the program but must be declared.

Declaration is done as follows:

<VariableName>: Datai <InitialValue>

where <InitialValue> is the initial value of the variable.

e.g. MyVariable: Datai 48

Would declare a variable called MyArray with a value of 48.

Used:

<Instruction>  <Register>, MyVariable

e.g. Dvd Indx, Apples ;Divide contents of indx with contents of apples

 

Immediate Addreessing

This is where the value to be used with an instruction is given explicitly.

Used:

<Instruction>  <Register>,#<Integer>

e.g. Add Indx,#23 ;Adds 23 to the index register

 

Rnadom Function

This is a special function that returns a random integer.  It is similar to the way arrays are used but does not require any declaration.

LOAD Acc,Rnd(<Real>)
LOAD Acc,Rnd(<Register>)

Where <Real> is a positive integer and register <Register> = Acc | Indx

e.g. Load Acc,Rnd(3) ;Load into accumulator random number between 0 and 2.
