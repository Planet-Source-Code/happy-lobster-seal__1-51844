Add <register>, <address>

Add contents of address into register.  Set Flag register according to result.

<register> = acc | indx

Clrs <address>

Clear the screen with the colour value of the contents of address.

Valid colour values are 0-15
Address is optional.

Clrz <register>

Zeroise contents of register.

<register> = acc | indx | flag

Cmpr <register>, <address>

Compare contents of address with register.  Set Flag register according to result.

<register> = acc | indx

Copy <register>, <address>

Copy contents of register into address.

<register> = acc | indx

Dec <register>

Decrement register.  Set Flag register according to result.

<register> = acc | indx

Dvd <register>, <address>

Divide contents of register with address.  Set Flag register according to result.

<register> = acc | indx

Exit

Return control from subroutine.

Halt

Terminate program execution.

Inc <register>

Increment register.  Set Flag register according to result.

<register> = acc | indx

Inpti <device>, <address>

Input integer from device to address.

<device> = kbd

Jeqz <label>

Jump to label if register is zero.

Jgez <label>

Jump to label if register is greater than or equal to zero.

Jgtz <label>

Jump to label if register is greater than zero.

Jlez <label>

Jump to label if register is less than or equal to zero.

Jltz <label>

Jump to label if register is less than zero.

Jnez <label>

Jump to label if register is not zero.

Jsubr <label>

Jump to label subroutine call.

Jump <label>

Jump to label.

Load <register>, <address>

Load contents of address into register.   Set Flag register according to contents loaded.

<register> = acc | indx

Mod <register>, <address>

Calculate the modulus of register with address.  Set Flag register according to result.

<register> = acc | indx

Mpy <register>, <address>

Multiply contents of address with register.  Set Flag register according to result.

<register> = acc | indx

Neg <register>

Negate register.   Set Flag register according to result.

<register> = acc | indx

Ouptc <device>, <address>

Set the ouput text colour.  Set Flag register according to result.

Valid colour values are 0-15
<device> = scr

Oupti <device>, <address>

Output integer from address to device.

<device> = scr

Oupts <device>, ' <string> '

Output string to device.

<string> = a sequence of characters

Sub <register>, <address>

Subtract contents of address from register.  Set Flag register according to result.

<register> = acc | indx
