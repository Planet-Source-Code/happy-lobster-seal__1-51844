Machine Components:

Control Unit

The control unit handles execution of a program. It fetches instructions from memory and executes them by dispatchng the relevant control or data signals to the appropriate component.

Arithmetic and Logic Unit

The ALU carries out arithmetic operations and logic comparisons. It contains an accumulator, index, and flag register, which temporarily store the result of the last arithmetic computation. After every arithmetic operation the flag register is set, which can be positive, zero, or negative, depending on the result of the last instruction. The ALU only operates in the range -32768 to +32767.

Memory

There is enough space in the machine's memory for:
256 integer variables
256 1 dimensional arrays with maximum size of 256 integer elements
Unlimited line labels and instructions

Colour Screen

The colour screen is where integers and strings are output. There are 16 basic colours available and support for full screen program output.

The colours values are:

0 Black 8 Grey
1 Blue 9 Light Blue
2 Green 10 Light Green
3 Cyan 11 Light Cyan
4 Red 12 Light Red
5 Magenta   13 Light Magenta
6 Yellow 14 Light Yellow
7 White 15 Bright White

Keyboard

This allows integers to be input into the system. You must press return after typing a number to allow execution of your program to continue.
